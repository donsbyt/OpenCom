# OpenCom Admin Panel

Standalone platform admin panel (separate from the main `frontend/` app).

## Run locally

```bash
cd panel
npm install
npm run dev
```

Default dev URL: `http://localhost:5175`

## Env

- `VITE_CORE_API_URL`: Core API base URL
- `VITE_SERVER_ADMIN_URL`: Link target for the server-admin page
